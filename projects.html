My CS Projects

Some of the my favorite projects I've worked on since first learning Computer Science are listed below

https://github.com/EmileJB/Hexmap-Converter

This project was designed so that I could extract desired information from Worldographer's (a hex map making program) unique .wxx files and create a JSON from it that is used in the project below this one.

https://emilejb.github.io/compass/ (Website)

https://github.com/EmileJB/EmileJB.github.io/tree/master/compass (Repo)

Although never finished and no longer in use, this website was designed so that players in my D&D campaign would be able to keep track of what they've done and where they've been so far.The map is made using JavaScript and HTML5's canvas.

https://github.com/EmileJB/ethics-final

My final project for the Ethics in Computer Science course. This project uses the Transitland API to get how many public transportation options are around a certain location (with a radius specified by the user). We used this program to investigate various bus stops around NYC in order to show how limited people's transportation options were at various parts of the city.

https://github.com/EmileJB/pd9_Comp-Scike_EmileJB-MatthewC-AaronM

My final project for AP Computer Science 10 years ago. It was a tower defense game based off various characters fighting against the (now retired) bugs of GridWorld. While not technically impressive, I think it would be fun to show off to students in AP CS A classes in the future the project I made when I was in AP CS, and talk about how the class has changed since then.
